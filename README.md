## Resumen del Proyecto

El proyecto **STARTRAINING** consiste en el desarrollo de un sistema web orientado a la gestión de convocatorias de prácticas preprofesionales, con el objetivo de facilitar la conexión entre empresas y estudiantes de educación superior.

La plataforma permite a las empresas publicar convocatorias especificando los requisitos, perfiles solicitados y condiciones de las prácticas. Por otro lado, los estudiantes pueden acceder libremente al sistema, visualizar las oportunidades disponibles y postular enviando sus datos y su currículum vitae.

El sistema fue desarrollado utilizando tecnologías web como **PHP para el backend**, **PostgreSQL como gestor de base de datos** y herramientas de automatización como **n8n** para integrar y optimizar procesos dentro de la plataforma.

Asimismo, el sistema implementa una **arquitectura de tres capas**, que permite separar la interfaz de usuario, la lógica de negocio y la gestión de datos, garantizando mayor organización, seguridad y escalabilidad.

Finalmente, el sistema busca centralizar las convocatorias de prácticas en una sola plataforma digital, mejorando la accesibilidad para los estudiantes y optimizando los procesos de selección para las empresas.
